# 🌙 Perfil de Alunos - Perfil Interativo

Este projeto foi desenvolvido como atividade prática para a disciplina de **Programação Web II** no IFCE. Consiste em uma galeria de perfis de alunos com foco em um design editorial de alto luxo, utilizando tecnologias front-end puras.

## 🚀 Tecnologias Utilizadas
* **HTML5**: Estruturação semântica.
* **CSS3**: Estilização avançada com variáveis, Flexbox e Grid.
* **JavaScript (Vanilla)**: Lógica de interatividade, manipulação de DOM e leitura de arquivos.
* **Google Material Icons**: Biblioteca de ícones vetoriais.

## 🎨 Design System
O projeto segue o guia **"Editorial Dark Mode"**, com os seguintes princípios:
* **No-Line Philosophy**: A separação dos elementos é feita por tons de superfície, sem o uso de bordas de 1px.
* **Nocturnal Palette**: Tons profundos como `#0E0E0E` contrastando com cyans vibrantes.
* **Typography**: Uso exclusivo da fonte **Plus Jakarta Sans** para um ar moderno e geométrico.

## 🛠️ Funcionalidades
* **Alternância de Tema**: Botão global para trocar entre Modo Claro e Modo Escuro (Dark Mode por padrão).
* **Paleta Cromática**: Alteração dinâmica da cor de fundo dos cards e elementos internos (halo, cargo e tags).
* **Sistema de Seguidores**: Contador interativo que incrementa/decrementa e altera o estado do botão.
* **Upload de Avatar**: Possibilidade de trocar a foto de cada perfil individualmente através de um seletor de arquivos.
* **Janela Modal**: Visualização detalhada de cada aluno ao clicar no botão "Ver Perfil".

## 📱 Responsividade
O layout foi projetado para ser totalmente responsivo:
* **Desktop**: Grid de 3 colunas.
* **Tablet**: Grid de 2 colunas.
* **Mobile**: Layout vertical em coluna única.
